# Alphabet and vowel sounds

Optional MP3 files for the Hebrew Basics screen. If a file is present, it is played when the user taps that letter or vowel; otherwise the app uses Hebrew TTS.

## Letters (`alphabet/`)

File name = letter name in lowercase with spaces as underscores, plus `.mp3`.

Examples: `alef.mp3`, `bet.mp3`, `gimel.mp3`, `kaf_sofit.mp3`, `tsadi_sofit.mp3`.

Full list (from `lib/data/hebrew_alphabet.dart`): alef, bet, gimel, dalet, he, vav, zayin, het, tet, yod, kaf, lamed, mem, nun, samekh, ayin, pe, tsadi, qof, resh, shin, tav, kaf_sofit, mem_sofit, nun_sofit, pe_sofit, tsadi_sofit.

## Vowels (`vowels/`)

File name = vowel name in lowercase with spaces as underscores, plus `.mp3`.

Examples: `patach.mp3`, `kamatz.mp3`, `shuruk.mp3`, `kubutz.mp3`, `shva.mp3`.

Full list: patach, kamatz, tzere, segol, hirik, holam, shuruk, kubutz, shva.
